# Tic Tac Toe . version consola
Este proyecto implementa un juego de Tres en raya en consola desarrollado en java. Dos jugadores se turnan para colocar sus simbolos (X y O) en un tablero 3x3 hasta uno gana o termine en empate.

## Caracteristicas
- Juegos de tres en raya en consola
- Partida para dos jugadores
- Deteccion automatica de victoria
- Deteccion de empate
- validacion de jugadas
- visualizacion de tablero en consola
## Tecnologias utilizadas 
- Java
- GitHub

### Requisitos previos
- java 18 o superior
